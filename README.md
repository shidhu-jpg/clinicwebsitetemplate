# Clinical Precision Website Template

A premium diagnostic clinic website template featuring modern design and responsive layout.

## Pages

- **index.html** - Homepage with hero section and services overview
- **services.html** - Detailed services page
- **anout.html** - About us page

## Features

- Responsive design with Tailwind CSS
- GSAP animations
- Modern glass-morphism effects
- Premium typography with Manrope and Inter fonts
- Material Design icons

## GitHub Pages

This repository is configured for GitHub Pages. Visit your live site at: `https://shidhu-jpg.github.io/clinicwebsitetemplate/`

## Technologies Used

- HTML5
- Tailwind CSS
- GSAP (GreenSock Animation Platform)
- Material Symbols
- Google Fonts
